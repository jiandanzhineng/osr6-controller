# OSR6 UDP 控制器 (带缓和功能)

这是一个简单的桌面程序，整合了 `udp_server.py`、`serial_sender.py` 和 `osr6.py` 的功能。

## 主要功能

1.  **UDP 接收**：模拟 OSR6 设备，监听 UDP 端口（默认 8080），接收 TCode 指令。
2.  **缓和处理**：内置自适应轴范围算法，对接收到的 TCode 进行平滑处理（缓和功能）。
3.  **串口发送**：将处理后的指令通过串口发送给实际硬件。
4.  **GUI 界面**：
    *   可视化选择串口端口。
    *   可调节缓和系数（缩放比例）。
    *   实时显示运行日志。

## 使用方法

1.  **运行程序**：
    ```bash
    python main_gui.py
    ```

2.  **操作步骤**：
    *   **选择串口**：在下拉框中选择设备连接的 COM 口。如果没有看到，点击“刷新”按钮。
    *   **设置 UDP 端口**：默认为 8080，通常无需修改。
    *   **调节缓和参数**：拖动滑块调整动作幅度缩放比例（0.0 - 1.0）。
    *   **启动服务**：点击“启动服务”按钮。

3.  **状态监控**：
    *   下方的日志窗口会显示连接状态和收到的指令（为了性能，高频 TCode 不会全部打印）。

## 依赖库

程序使用了 Python 标准库 `tkinter` 和 `socket`，以及第三方库 `pyserial`。
如果未安装 `pyserial`，请运行：
```bash
pip install pyserial
```
